# Sandbox

## Dependencies
* SDL
    * `sudo apt-get install libsdl2-dev`
* LuaJIT
    * `git clone https://github.com/LuaJIT/LuaJIT.git`
    * `cd LuaJIT`
    * `make && sudo make install`

## Setup
    git clone --recursive ${REPOSITORY_URL}
    cd ${REPOSITORY_NAME}
    chmod +x build.sh
    ./build.sh
